Python 基础学习 - 第4天：循环遍历列表

你好！在之前的学习中，我们如果想对列表里的每个元素都做点什么（比如打印出来），我们得一个一个地手动去访问，像这样：

friends = ["张三", "李四", "王五"]
print(friends[0])
print(friends[1])
print(friends[2])

如果列表有一百个朋友呢？这显然不现实。今天，我们将学习一个强大的工具来解决这个问题：for 循环。

1. 什么是 for 循环？​

for 循环可以"遍历"一个序列（比如列表）中的每一个元素，并对每个元素执行相同的代码块。

它的基本语法结构如下：

for item in list_of_items:
    # 对 item 执行某些操作
    # 这部分代码会重复执行
  • for ... in ...: 是固定的语法。
  • list_of_items: 你想要遍历的列表变量。
  • item: 这是一个临时变量，Python会在循环的每一次迭代中，将列表中的一个元素按顺序赋值给它。你可以给这个变量取任何有意义的名字。
  • 缩进: for 循环下面所有缩进的代码行都属于循环体，它们会为列表中的每个元素重复执行。缩进在Python中是强制的，它定义了代码块。

示例：向所有朋友问好​

让我们用 for 循环来重写最开始的例子：

friends = ["张三", "李四", "王五"]

for friend in friends:
    print(f"你好，{friend}！很高兴见到你。")

print("循环结束了！") # 这行没有缩进，所以它在循环结束后只执行一次。

代码是如何工作的？

  1. Python从 friends 列表中取出第一个元素 "张三"，并将它赋值给临时变量 friend。
  2. 执行缩进的代码块：print(f"你好，张三！很高兴见到你。")。
  3. 接着，取出第二个元素 "李四"，赋值给 friend。
  4. 再次执行缩进的代码块：print(f"你好，李四！很高兴见到你。")。
  5. ...直到列表中所有元素都被访问过一遍，循环结束。
  6. 执行循环后面的代码 print("循环结束了！")。

2. 在循环中进行计算​

你可以在 for 循环内进行任何操作，包括数学计算。

numbers = [1, 2, 3, 4, 5]
total = 0 # 先初始化一个变量用来存储总和

for number in numbers:
    total = total + number # 每次都把当前数字加到总和上

print(f"所有数字的总和是: {total}") # 输出: 所有数字的总和是: 15

3. 使用 range() 函数创建数字序列​

有时，你可能需要循环一个固定的次数，而不是遍历一个现有的列表。这时 range() 函数就派上用场了。

  • range(5): 会生成一个从 0 到 4 的序列 (0, 1, 2, 3, 4)。
  • range(1, 6): 生成一个从 1 到 5 的序列 (1, 2, 3, 4, 5)。第一个数是起始值（包含），第二个数是结束值（不包含）。

range() 经常和 for 循环一起使用：

for i in range(5):
    print(f"这是第 {i} 次循环")

你也可以使用 range() 来创建数字列表：

number_list = list(range(1, 6))
print(number_list) # 输出: [1, 2, 3, 4, 5]

4. 列表推导式 (List Comprehensions) - 进阶技巧​

Python提供了一种更简洁的方式来基于一个序列创建新的列表，叫做列表推导式。

假设我们想创建一个包含数字1到5的平方的列表。

传统方法 (使用 for 循环):

squares = [] # 创建一个空列表
for value in range(1, 6):
    square = value ** 2
    squares.append(square)
print(squares) # 输出: [1, 4, 9, 16, 25]

使用列表推导式:

squares = [value**2 for value in range(1, 6)]
print(squares) # 输出: [1, 4, 9, 16, 25]

这种写法将 for 循环和创建新元素的代码合并到了一行，非常高效和 Pythonic。初学时你可能觉得有些复杂，可以先了解，优先掌握标准的 for 循环。

for 循环是编程自动化处理任务的核心，请务必多加练习，真正理解它的工作原理。
