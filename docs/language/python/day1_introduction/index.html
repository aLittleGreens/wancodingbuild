Python 基础学习 - 第1天：初识Python与变量

欢迎来到Python学习的第一天！今天我们将从最基础的概念开始，带你敲开编程世界的大门。

1. 什么是Python？​

Python是一门非常流行的高级编程语言。它之所以受欢迎，是因为它的语法非常简洁、清晰，就像是在写英文句子一样，所以被称为"可读性高"的语言。

  • 优点: 简单易学、功能强大、应用领域广泛（我们之前讨论过的Web开发、数据科学等）。
  • 用途: 你可以用它来制作网站、分析数据、自动完成任务，甚至开发游戏和人工智能。

2. 准备你的学习环境​

在开始写代码之前，你需要两样东西：

  • Python解释器: 这是让你的电脑能够理解并执行Python代码的核心程序。
    • 请访问 python.org 下载并安装最新稳定版的Python。
    • 重要提示: 在Windows上安装时，请务必勾选 "Add Python to PATH" 这个选项，这样会方便很多。
  • 代码编辑器 (IDE): 这是你用来编写、管理和运行代码的软件。
    • 推荐: Visual Studio Code (VS Code) 或 PyCharm Community Edition。它们都有很好的Python支持，能提供代码高亮、错误提示等功能，极大地提升你的编程体验。

3. 你的第一个Python程序: "Hello, World!"​

学习任何一门编程语言的传统都是从打印 "Hello, World!" 开始。这能帮助你验证环境是否配置正确。

打开你的代码编辑器，新建一个文件（例如 hello.py），然后输入以下代码：

print("Hello, World!")
  • print() 是Python内置的一个函数，它的作用就是把括号里的内容输出到屏幕上。
  • "Hello, World!" 是一个字符串，也就是一段文本。在Python中，文本需要用英文的引号（单引号 ' ' 或双引号 " "）包围起来。

在你的编辑器或终端中运行这个文件，你应该能看到屏幕上打印出了 Hello, World!。

4. 注释：给代码添加说明​

注释是代码中不会被执行的部分，它们是写给人看的，用来解释代码的功能或逻辑。在Python中，以 # 号开头的任何内容都是注释。

# 这是一行注释，Python会忽略它
print("这条语句会被执行")  # 注释也可以写在代码行的末尾

养成写注释的好习惯非常重要，它能帮助你和他人更好地理解你写的代码。

5. 变量和基本数据类型​

变量 (Variable) 可以看作是一个用来存储数据的"盒子"，你可以给这个盒子起一个名字，然后把数据放进去。

# 创建一个名为 message 的变量，并把字符串 "你好，Python！" 存进去
message = "你好，Python！"

# 打印变量中存储的内容
print(message)

变量的名字（比如 message）应该有意义，让人一看就知道它存储的是什么。变量名只能包含字母、数字和下划线 _，且不能以数字开头。

今天我们先接触三种最基本的数据类型：

  • 字符串 (String): 用来表示文本，需要用引号包围。
    • name = "张三"
  • 整数 (Integer): 就是不带小数点的数字。
    • age = 25
  • 浮点数 (Float): 就是带小数点的数字。
    • height = 1.75

Python是动态类型语言，你不需要预先声明变量的类型，解释器会自动识别。

name = "李四"      # 这是一个字符串 (str)
age = 30         # 这是一个整数 (int)
pi = 3.14        # 这是一个浮点数 (float)

print(name)
print(age)
print(pi)

今天的内容就到这里。请务必亲手实践，把上面的每一个代码示例都在你的编辑器里敲一遍并运行，感受一下它是如何工作的。然后，尝试完成我们为你准备的课后练习。
